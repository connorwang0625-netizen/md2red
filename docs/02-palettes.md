# 02 · 配色系统：单色相 + 60-30-10

> 结构骨架与配色正交（见 [01-design-system.md](./01-design-system.md)）。颜色不是「三选一固定盘」，而是**一篇一个强调色相、品牌锡点不变**的系统。
> 当前已实现的命名预设（`.theme-mint` / `.theme-acid` / `.theme-maillard`）见文末——它们是这套系统的**预设实例**，不是三套平级配色。

---

## 颜色模型（铁律）

### ① 文字不参与配色

- 文字走**独立中性墨阶**：`--c-ink`（近黑/近白主墨） / `--c-ink-2`（次级） / `--c-ink-3`（弱化·出处）。
- 中性墨**跨篇基本不变**，不随强调色相走。60-30-10 只管「面 + 强调」，不管文字。

### ② 60-30-10（只管面与强调）

- **60 主场** = `--c-page` + `--c-card`。近中性，**带极低饱和的微色**（非纯白/纯灰），有温度但不抢眼。
- **30 结构** = `--c-card-2` + `--c-hair` + 操作条描边。与 60 同向的近中性微色，但**明度跨一个看得见的台阶**（现状 `#F4F8FC→#EAF1F9` 台阶太小，需用 OKLCH 拉到感知可分辨）。
- **10 强调** = `--c-anchor`。每页**只钉一个重点**（≤10%，越少越好）：焦点面板锚点边 / 序号 chip / 单位 em / 操作图标 / 当前 dot。撒散即违例（接 N18 / UX-2）。

### ③ 强调色 = 每篇的「情绪色」（对冲）

- **10 是每篇唯一的选色决策**，按内容情绪/对比选一个色相（emotion→hue 见下，表待正式定）。
- **10 与 60/30 是对冲关系**：60/30 是近中性微色，10 用**对比 / 互补方向**的色相跳出——不是「整卡淡染同一色」，而是中性场上钉一点对冲色。
- 用 OKLCH 选 10（中等 L、较高 C），保证跨色相的强调强度一致。

### ④ 品牌锡点 = 结构 + 一个固定签名色触点

- 品牌识别**主要来自**：骨架 + 字体 + 中性墨 + 60-30-10 纪律 + dots / brandfoot 等结构签名 → 每篇换强调色，主页也不会千篇一律一个色。
- **外加一个极小的固定签名色触点**（跨篇不变）：落在**页眉期号 chip（`.h-left`）**一处（或脚注），用一个固定品牌色。
  - 这是「元信息走中性墨」（UX-2 / N18）的**唯一有意例外**：仅此一处、面积极小。
  - 它与每篇的情绪强调色 10 是**两个不同层**：签名色 = 固定品牌（元信息层），强调色 10 = 当篇情绪（内容焦点层）；两者不在同一元素上叠加。

### ⑤ acid = 暗底霾虹 wildcard（例外）

- `.theme-acid` 不走单色相 / 对冲规则：它是暗底 + 多色霾虹渐变的**意外款**，单列。大字用 600 重 + `drop-shadow` 发光（唯一与「越大越细」不同处）。

## emotion → hue（强调色 10 的选法，表待正式定）

| 情绪 / 主题 | 色相方向 |
|---|---|
| 平静 / 健康 | 蓝青 |
| 信任 | 蓝 |
| 温暖 / 食物 | 琥珀 / 陶土 |
| 能量 / 紧迫 | 橙红 |
| 柔媚 | 品红 |
| 生长 | 绿 |

## 当前已实现的命名预设（card-base.css）

以下三个 `body.theme-*` 是上面系统的**预设实例**，hex 已写死于 `templates/card-base.css`。

> **grandfather 说明**：现存 mint / maillard 是早期「60/30 向强调色同色相微染」的预设，与上面新的「近中性 60/30 + 对冲 10」模型略有出入。**HE-0001（首页冻结）用的是 `theme-mint`，保留不动**；新篇按新模型派生强调色。card-base.css 的 token 是否重做（改成 `--h` 驱动）= 单独的 CSS 步，动它会影响冻结的 HE-0001，暂不动。

### theme-mint · 蓝预设（Swiss 基线）

雾蓝白底 + 信任蓝强调。冷静、专业、可读。

| token | 色值 | 含义 |
|---|---|---|
| `--c-page` | `#D7E1EC` | 展厅底 |
| `--c-card` | `#F4F8FC` | 卡片底 |
| `--c-card-2` | `#EAF1F9` | 次级面 |
| `--c-ink` | `#161B22` | 主墨 |
| `--c-ink-2` | `#5A6675` | 次级 |
| `--c-ink-3` | `#8B97A6` | 弱化 / 出处 |
| `--c-anchor` | `#2D7FF9` | 锚点·信任蓝 |
| `--c-anchor-d` | `#1F6FE5` | 锚点深色 |
| `--c-hair` | `#C9D5E3` | 细线 |

### theme-maillard · 陶土预设

暖奶油底 + 赤陶/鼠尾草强调。柔和、静谧、去饱和。

| token | 色值 | 含义 |
|---|---|---|
| `--c-page` | `#E3DACB` | 展厅底·暖米 |
| `--c-card` | `#F3ECDF` | 卡片底·奶油 |
| `--c-card-2` | `#EADFCC` | 次级面 |
| `--c-ink` | `#2F2820` | 主墨·暖棕 |
| `--c-ink-2` | `#6E6253` | 次级·棕灰 |
| `--c-ink-3` | `#9C907E` | 弱化·出处 |
| `--c-anchor` | `#B05E3B` | 主锚点·赤陶 |
| `--c-anchor-2` | `#7E8C6A` | 次锚点·鼠尾草 |
| `--c-hair` | `#D6C9B4` | 细线 |

- 低对比、去饱和；严格遵守「越大越细」、无圆角无阴影。数据块左缘 4px 鼠尾草；数据单位 / 重点用赤陶。

### theme-acid · 霾虹 wildcard

炭黑底 + 多色霾虹渐变锡点。高能、年轻、抢眼。

| token | 色值 | 含义 |
|---|---|---|
| `--c-page` | `#0A0A0C` | 展厅底 |
| `--c-card` | `#141418` | 卡片底·炭黑 |
| `--c-card-2` | `#1E1E26` | 次级面 |
| `--c-ink` | `#F5F5F7` | 主墨·近白 |
| `--c-ink-2` | `#9A9AA8` | 次级 |
| `--c-ink-3` | `#62626E` | 弱化 |
| `--c-anchor` | `#FF2D9B` | 单色锡点（小件） |
| `--grad` | `linear-gradient(100deg,#C6FF3A 0%,#16E0FF 46%,#FF2D9B 100%)` | 霾虹渐变（大字/序号/数据） |
| `--c-hair` | `#2C2C36` | 细线 |

- 霾虹渐变仅用于 Display / Index / Num / 封面重点字 / recap 序号 / CTA 重点字，配 `drop-shadow` 软晕发光。
- 数据块左缘 4px 霾虹色边；单位 `em` 用纯电青 `#16E0FF`。
- 为发光可读，此版大字采用 600 重（唯一与「越大越细」不同之处）。
