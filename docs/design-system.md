# 设计系统

md2red 的视觉宪法。所有模板必须遵守本文件。

## 1. 两种风格（与内容类型解耦）

风格是「视觉姿态」，不是内容分类。任何选题都可二选一，取决于编辑意图，而不是按题材查表。

- **Swiss 国际主义** — 工程 / 量化的节奏。无衬线、严格网格、单一高饱和锚点、越大越细、直角模块、发丝线。适合数据 / 科普 / 清单 / 对比，但叙事内容也能用。
- **Editorial 杂志 × 电子墨水** — 慢节奏特写。衬线标题、纸张氛围、留白、栏目结构。（后续接入）

一套作品只能用一种风格，不可混用。

## 2. Swiss 纪律（硬规则）

1. **越大越细**：≥ 72px 的大字字重 ≤ 300；只有小字（标签 / meta）才用 500–650。
2. 不加载衬线字体；正文用 sans-zh。
3. 分隔只用**发丝线（1–2px）或栏沟**，禁卡片边框 + 阴影。
4. 全篇只有**一个锚点色**；禁混用多色。
5. **禁圆角、禁阴影、禁渐变**。
6. 标题落在**左上内容轴**，不居中（方形封面除外）。
7. 图片是证据块，直角、稳定比例容器。

> 通不过任一条，就不是 Swiss，是「带衬线的伪 Swiss」或通用卡片。

## 3. 类型刻度（1080×1440）

### Swiss

| 角色 | 字号 | 字重 |
| --- | --- | --- |
| Hero | 240px | 200 |
| Statement | 180px | 200 |
| 章节标题 h-xl | 96–120px | 300 |
| 中标题 h-md | 56px | 400 |
| 巨型数字 num-mega | 200px | 200 |
| 正文 body | 28–32px | 400 |
| 分类 t-cat | 22px | 600 |
| meta / 标签 | 20–21px | 500（mono）|

中文标题更密：先缩字数，再缩字号。`.h-xl` 在竖版上限约 2 行 × 8–9 字。

### 最小可读字号（移动端）

正文 28 / lead 30 / caption·kicker 20 / 标签 20 / 网格单元标题 24 / 数字注释 22。**放不下就删字，不缩字号。**

## 4. 间距 token（Carbon 2x）

`8 / 12 / 16 / 24 / 32 / 40 / 48 / 64 / 80 / 96 / 160`

- 竖版外框内边距 = **96**
- 区块间距 = **48（默认）/ 64（大段断）**
- 卡片内边距 = **40**

所有上下左右间距只从这套取，**禁随手 px**——随手 px 会让每页节奏漂移。

## 5. 配色系列

HSL 是**区间**不是固定值；用量比 **60-30-10**（底色 60 / 字色 30 / 锚点 ≤10）。

### B 清新薄荷（健康饮食 HE）
- 底：雾蓝白 `#F4F7F8`（或纯白 `#FFFFFF`）
- 字：石墨 `#1C2226`
- 锚点池（≤10%）：薄荷绿 `#3FB984` / 翡翠青 `#16B8A6` / 蒂芙尼青 `#25B4CC` / 天青蓝 `#2E9BDD` / **信任蓝 `#2D7FF9`**

### C 酸性 · 多巴胺霓虹
- 底：炭黑 `#17181A`
- 字：纸白 `#F5F1E8`
- 锚点池（≤5–8%）：能量橙 `#FF6A00` / 荧光黄绿 `#C6FF2E` / 赛博青 `#19E5FF` / 电光紫 `#7C4DFF` / 霓虹品红 `#FF2EA6`

### D 美拉德 × 莫兰迪
- 底：暖奶油 `#FBF4EA`（或 `#F3E7DD`）
- 字：暖棕 `#3A2E26`
- 锚点池（≤10%）：赤陶橙 `#C9744A` / 赭黄·芥末 `#C79A4B` / 陈玫瑰灰粉 `#C0988E` / 鼠尾草绿 `#8AA398` / 雾蓝 `#9DB4C0`

### 锚点取值边界
暖 H15–40 / S35–55 / L45–60；冷 H80–210 / S12–25 / L55–70。范围内自由选，看着舒服即可。
